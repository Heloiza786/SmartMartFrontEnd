## Passo a passo para utilização

1. Inicie baixando as dependências(certifique-se se o node.js já esta instalo):
```bash
npm install
```

2. Inicie o servidor com o comando:
```bash
npm run dev
```

3. Acesse de forma local:
```
http://localhost:5173/
```
